# 💬 Web Chat

**Web Chat** is a realtime messaging platform that enables users to communicate through **private and group conversations**.  
The system focuses on providing a scalable backend architecture for **realtime communication, user presence tracking, and media sharing**.

This project currently focuses on **backend development**, implementing the core services required for a modern chat application.

---

# ✨ Features

## 👥 Messaging
- Realtime **one-to-one messaging**
- **Group conversations**
- **Message history retrieval**
- Message synchronization through **Socket.IO**

## 🤝 Friend System
- Send and receive **friend requests**
- Users must **become friends before sending messages**
- **Remove friends**
- **Block users**

## 👤 User Management
- Update **user profile information**
- **Change password**
- Account management features secured with authentication

## 📎 File Sharing
- Upload and share files within chat messages
- Media files are stored using **Cloudinary**

## 🔔 Notifications
- Realtime notifications for:
  - New messages
  - Friend requests

## 🟢 User Presence
- Track **online/offline status**
- Presence information managed using **Redis**
